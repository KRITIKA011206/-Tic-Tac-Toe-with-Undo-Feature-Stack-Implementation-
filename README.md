# -Tic-Tac-Toe-with-Undo-Feature-Stack-Implementation-
This project is a **console-based Tic Tac Toe game in C** that demonstrates the use of the **Stack data structure** to implement an **Undo functionality**.

##  Features
- Two-player Tic Tac Toe game (X vs O)
- Stack-based undo operation
- Input validation for moves
- Winner and draw detection
- Clean and simple console interface

##  Concepts Used
- Stack (Array implementation)
- Structures in C
- 2D Arrays
- Modular programming
- Game logic

##  How to Run

### Compile
```bash
gcc tic_tac_toe.c -o tic_tac_toe
